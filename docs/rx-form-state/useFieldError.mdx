# useFieldError

获取表单域验证错误状态的 hook。

```tsx
import { useFieldError } from '@sinoui/rx-form-state';

function Demo() {
  const errorMsg = useFieldError('userName');

  return <div>表单域验证错误：{errorMsg}</div>;
}
```
